    # 搭建vue开发环境

    vue create hello-world

    如果你还没有安装vue-cli，请先安装，或者你用你的方法搭建好之后进行下一步

    # 安装Zing-UI

    # 安装

    yarn add zing-ui
#or
npm install zing-ui --save

    # 样式引入

    在入口文件引入css样式:

    import 'zing-ui/dist/index.css'

    # 注册组件

    在组件中引入

    <script>
import {Button} from 'zing-ui'
export default {
  name: 'yourApp',
  components: {
    Button
  }
}
</script>

    # 使用

    <Button type="success">一个的按钮</Button>
