Zing-UI

一个基于Vue + TypeScript + Sass 的UI组件库

快速上手 →

Vue

为Vue的过大使用者提供方便快捷的UI库，易于上手，还便于与第三方库或既有项目整合。

TypeScript

TypeScript 发展至今，已经成为大型项目的标配，其提供的静态类型系统，大大增强了代码的可读性以及可维护性。

Sass

Sass 是一种 CSS 的预编译语言，能够帮助复杂的样式表更有条理， 并且易于在项目内部或跨项目共享设计。
