# 基本信息
Zing-UI 是一个基于Vue2 + TypeScrtpt + Sass 的UI组件库，涵盖了Button、Icon、TextBox、Tabs、Collapse、Popover等组件
# 开发目的
有人总说不要重复造轮子, 因为凭借自己的力量几乎不可能造出一个比业界共用的轮子还好的轮子, 造再多也是白费力。
这的确有点道理，但我的目的只是为了学习与提高，在这个过程中遇到的问题学习的东西都是很重要的。
